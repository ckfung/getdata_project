## Overview
The script "run_analysis.R" works as below:

###Pre-requisite:
* Download and unzip the data from https://d396qusza40orc.cloudfront.net/getdata%2Fprojectfiles%2FUCI%20HAR%20Dataset.zip 
* Rename the extracted folder to 'dataset'
* Make sure the 'run_analysis.R' script is at the same folder as 'dataset'

###Steps:
* Navigate your RStudio working directory to the same directory as above.
* Run the R script in RStudio using source("run_analysis.R")
* Once the script finish, it will generate tidydata.txt file
