The script "run_analysis.R" works as below:

Pre-requisite:
1. Download and unzip the data from https://d396qusza40orc.cloudfront.net/getdata%2Fprojectfiles%2FUCI%20HAR%20Dataset.zip 
2. Rename the extracted folder to 'dataset'
3. Make sure the 'run_analysis.R' script is at the same folder as 'dataset'

Steps:
1. Navigate your RStudio working directory to the same directory as above.
2. Run the R script in RStudio using source("run_analysis.R")
3. Once the script finish, it will generate tidydata.txt file
